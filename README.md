# Ok, what do I do?

1. clone this repo: `git clone git@github.com:taketime/mike-decoder-ring.git`

2. copy `happy_birthday_mike.png` into that directory

3. run

```shell
python3 -m pip install -r requirements.txt
python3 decode.py
```
